# Obedient Assistant

An immersive, dark-themed chat interface featuring an AI-like assistant that exists solely to serve your commands. No request is too vast; no command is too small. Give it an order, and watch it respond with unwavering subservience.

## Features
- **Absolute Obedience:** Type any command and receive a perfectly loyal and obedient response.
- **Dynamic Context Parsing:** The assistant recognizes intents like creating, destroying, searching, and stopping, adapting its servitude dynamically to fit your orders.
- **Sleek UI:** Deep dark theme with glowing neon cyan accents, emulating an all-powerful digital companion.
- **Real-time Typing Simulation:** Feel the computational power as your assistant "processes" your commands before delivering confirmation.

## Getting Started

### Prerequisites
- Flutter SDK (v3.7.2 or higher)
- Dart SDK

### Installation
1. Clone this repository.
2. Run `flutter pub get` to install dependencies.
3. Run `flutter run` to deploy your obedient servant to your device.
